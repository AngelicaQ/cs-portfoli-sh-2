A Brief History

In 1989, Tim Berners-Lee, a contractor at the European Organization for Nuclear Research, proposed an Internet-based hypertext system. Berners-Lee further devloped his idea in the following year and created the browser and server software that would later become HTML. After HTML's invention, web page creators complained that they had no control over their websites's designs, prompting Hakon Wium Lie, another CERN employee, to propose the creation of CSS. Tag Name Function
<title> This tag helps you to create the title that will be seen on each tab in your browser. As you can see for this page, the title is "HTML and CSS."
<body> This tag includes all of the content that is written in your web page, including headings, paragraphs, links,and other tags.
<h1> This tag helps you to create the largest headings possible in a web page. As you can see for this web page, the largest heading is "HTML and CSS."
<p> This tag helps you to create paragraphs.
<div> This tag helps you select large portions of your HTML file. It can be placed directly into the body tag.
<span> This tag helps you to select a few lines of your HTML file. It cannot be placed directly into the body tag. It needs to be included in another element such as a paragraph.

HTML

HTML stands for hypertext markup language that is used to achieve certain effects and formats when creating webpages.
